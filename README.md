# 🛍️ Retail Sales Forecasting & Inventory Optimization

## 📌 Overview
This project predicts retail sales and optimizes inventory using Machine Learning and data analysis.

It helps businesses:
- Forecast future demand 📈
- Avoid stockouts ❌
- Reduce excess inventory 📦
- Make better inventory decisions 💡

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit

---

## 🚀 Features
- 📊 Sales forecasting using Random Forest
- 📦 Inventory optimization (Safety Stock & Reorder Point)
- 🔄 Multi-product dataset simulation
- 📈 Actual vs Predicted sales visualization
- 📊 Product-wise sales bar chart
- 🎛️ Interactive dashboard with filters
- ⚠️ Reorder alerts system

---

## 🏗️ Project Workflow
1. Dataset generation (synthetic retail data)
2. Data preprocessing
3. Feature engineering
4. Machine learning model training
5. Sales prediction
6. Inventory optimization
7. Dashboard visualization

## ▶️ How to Run

### Step 1: Generate Dataset
python src/data_generator.py

### Step 2: Run Main Pipeline
python main.py


### Step 3: Run Dashboard
streamlit run app/dashboard.py
